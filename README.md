# 📸 Instagram Clone V2

A responsive, feature-rich Instagram clone built with vanilla HTML, CSS, and JavaScript. This version demonstrates professional project structure with separated files and additional features like Dark Mode and Stories.

## ✨ About the Project

**Instagram Clone V2** is a frontend simulation of the popular social media platform. Developed using modular HTML, CSS, and ES6 JavaScript, this application showcases modern web development practices. Key features include a dynamic feed, interactive stories bar, real-time post creation, and a theme switcher. It serves as a practical demonstration of DOM manipulation, state management, and responsive design.

## 🚀 Features

- **📱 Mobile-First Design:** Optimized layout that looks great on phones and desktops.
- **📖 Stories Bar:** Horizontal scrolling stories section at the top.
- **🌙 Dark Mode:** Toggle switch to instantly change the app theme.
- **📸 Create Posts:** Upload images and add captions directly from your device.
- **❤️ Interactive Feed:** Like posts by clicking the heart or double-clicking the image.
- **🧩 Modular Code:** Clean separation of HTML, CSS, and JavaScript files.

## 🛠 Tech Stack

- **HTML5:** Semantic structure.
- **CSS3:** Flexbox, CSS Variables (for theming), Transitions, Media Queries.
- **JavaScript (ES6):** DOM manipulation, Event Listeners, Array methods.
- **FontAwesome:** For icons.
- **Google Fonts:** 'Grand Hotel' and 'Roboto'.
